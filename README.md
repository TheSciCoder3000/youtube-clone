# YouTube Clone
It is a web clone of the popular video sharing and social media platform, YouTube. It is built using the React Library with a Google and YouTube API to allow its users to search for videos similar to the real YouTube Website.


## Features

- Sign In with your Google Account using Google OAuth2.0
- Search for YouTube videos using the YouTube API
- Fully Responsive Design


## Installation

### Clone the Repository
```sh
git clone https://github.com/TheSciCoder3000/youtube-clone.git
```

### Install the node modules
```sh
cd youtube-clone
npm install
```

### Start the Local Server
```sh
npm start
```
